Prototype calculation rules: digital reference thermometer dropdown fills the cal due date automatically; correction and reference maths, Display cycle, Display from product and Room Temperature average all rounded to 1 decimal place using improved robust mathematical rounding across all fields. Select the probe from the dropdown; the nearest correction to the Initial Set Point is used for As Found, and the nearest correction to the Final Set Point is used for As Left. Display from product is calculated from Display cycle Max and Min. Air and Load reference averages each use the highest corrected maximum and the lowest corrected minimum within that column; those two values are visibly marked in the table. Adjustment not needed is applied only when the As Found Air and Load results are both present and within the tolerance for this device (±0.5 °C). If either result is missing or outside tolerance, As Left remains active and must be completed.
